EXE模板管理

使用说明

在任意有Python环境的Windows机器上，运行客户端目录中的 build_exe.py 打包为EXE， 然后将 FileBackupClient.exe 上传到此处。之后即可一键生成含配置的EXE安装包。

检查EXE模板状态...

EXE部署步骤

  1. 点击左侧 "下载EXE安装包" 获取zip
  2. 解压到目标Windows机器任意目录
  3. 双击 FileBackupClient.exe 即可运行
EXE特性
  • 单文件独立运行，无需安装Python
  • 后台静默运行，无控制台窗口
  • 自动注册开机自启动
  • config.ini 放在EXE同目录即可

Python源码部署 (备选)

Windows
  1. 解压安装包到目标机器
  2. 双击 install.bat 运行
Linux
  1. 解压: unzip file_backup_client.zip
  2. 运行: bash install.sh

系统要求: Python 3.8+, 依赖: watchdog, psutil
